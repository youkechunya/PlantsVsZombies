# 🌱 PlantsVsZombies —— 植物大战僵尸（Unity 版）
一个基于 Unity 引擎开发的简易植物大战僵尸 Demo，实现了核心玩法的基础逻辑，适合学习与二次开发。

---

## ✨ 项目简介
本项目是《植物大战僵尸》的简化复刻版，目前已完成基础框架与核心玩法原型，可用于：
- Unity 2D 游戏开发学习
- 塔防游戏逻辑设计实践
- 代码结构与游戏流程的二次开发

> ⚠️ 当前为 **早期开发版本**，仅实现了小部分核心逻辑，完整玩法仍在完善中。

---

## 🎮 已实现功能
- ✅ 基础场景搭建（草地、行/列网格布局）
- ✅ 植物种植逻辑（点击选择 → 放置到格子）
- ✅ 僵尸生成与移动逻辑
- ✅ 简单的碰撞与伤害判定
- ✅ 阳光资源系统（基础版）

## 🚀 快速开始
### 1. 克隆项目
```bash
git clone https://github.com/youkechunya/PlantsVsZombies.git
```

### 2. 打开项目
- 启动 Unity Hub → 点击「打开项目」→ 选择克隆后的 `PlantsVsZombies` 文件夹
- 等待 Unity 导入资源与编译代码

### 3. 运行游戏
- 打开 `Assets/Scenes/MainScene.unity`（或对应主场景文件）
- 点击 Unity 编辑器顶部的「播放」按钮，即可开始游戏

## 📂 项目结构
```
PlantsVsZombies/
├── Assets/                 # Unity 资源目录
│   ├── Scripts/           # 核心逻辑代码
│   │   ├── Plant/         # 植物相关脚本
│   │   ├── Zombie/        # 僵尸相关脚本
│   │   ├── GameManager/   # 游戏管理与流程控制
│   │   └── UI/            # 界面交互脚本
│   ├── Sprites/           # 游戏图片资源
│   ├── Prefabs/           # 预制体（植物、僵尸等）
│   └── Scenes/            # 场景文件
├── .gitignore             # Git 忽略文件（Unity 标准模板）
└── README.md              # 项目说明文档
```
